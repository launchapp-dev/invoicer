# Strategic Questions

Questions the org should be asking. Generated by the question-generator agent from scanning the knowledge base.

## Lifecycle

1. **open** — question generated, needs investigation
2. **investigating** — someone is working on answering it
3. **answered** — answer documented in the file, ready for action extraction
4. **actioned** — action items created from the answer

## File Format

Each question is its own .md file named descriptively (e.g., `should-we-sunset-launchpad-baas.md`).

# Strategic Questions

Questions the org should be asking. Generated by the question-generator agent from scanning the knowledge base.

## Lifecycle

1. **open** — question generated, needs investigation
2. **investigating** — someone is working on answering it
3. **answered** — answer documented in the file, ready for action extraction
4. **actioned** — action items created from the answer

## File Format

Each question is its own .md file with YAML frontmatter. Named with kebab-case describing the question (e.g., `should-we-sunset-launchpad-baas.md`).

### Template

```markdown
---
title: "Should we sunset the LaunchPad BaaS SDKs?"
priority: high
status: open
category: product-strategy
source_files:
  - knowledge/products/01-launchpad-baas.md
  - knowledge/revenue/opportunities.md
generated_by: question-generator
generated_at: 2026-03-19
---

## Context

Why this question matters right now.

## What We Know

Relevant facts from the knowledge base that inform this question.

## What We Don't Know

The specific gap this question exposes.

## Suggested Investigation

How to find the answer — data to gather, people to ask, experiments to run.

## Answer

_To be filled in by the team or an investigating agent._
```

### Frontmatter Fields

| Field | Type | Required | Values |
|-------|------|----------|--------|
| `title` | string | yes | The question itself |
| `priority` | enum | yes | `critical`, `high`, `medium`, `low` |
| `status` | enum | yes | `open`, `investigating`, `answered`, `actioned` |
| `category` | enum | yes | `product-strategy`, `technical-debt`, `competitive`, `revenue`, `architecture`, `operations`, `market-trends`, `security` |
| `source_files` | list | yes | Knowledge files that informed this question |
| `generated_by` | string | yes | Agent that created it |
| `generated_at` | date | yes | ISO date |
| `answered_by` | string | no | Who answered (human name or agent) |
| `answered_at` | date | no | When answered |

## Context

The vision doc identifies AO's competitive differentiation as "local-first, BYOK, multi-model orchestration with worktree isolation and self-healing." Competitors listed are Devin ($500/mo), Factory, and Sweep. However, GitHub Copilot Workspace is already in preview, and Cursor has composer mode. If any of these well-funded competitors adds comparable agent orchestration features, AO's differentiation could evaporate quickly. The current "moat" is a feature set, not a network effect or data advantage.

## What We Know

- AO's current advantages: BYOK (bring your own keys), local-first, worktree isolation, multi-model routing
- Devin pricing: $500/mo (far above AO's $29-49 target)
- Factory and Sweep are YC-backed competitors with funding
- AO has no proprietary model — it orchestrates third-party LLMs
- No documented switching costs or data lock-in for AO users
- AO skills/packs ecosystem is nascent (15 skills, ~15 packs)

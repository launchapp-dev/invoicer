# Quality Audit Reports

Build/test/lint results from actually running code in repos. Generated by the quality-runner agent.

## File Format

Each report is `<repo>-audit.md` with YAML frontmatter:

```markdown
---
repo: saas-template-launch-app-test
date: 2026-03-19
build_status: pass|fail
test_status: pass|fail|skip
lint_status: pass|fail|skip
---

## Results
...
```

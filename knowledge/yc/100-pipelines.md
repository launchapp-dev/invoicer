# 100 Pipelines AO Can Orchestrate

AO is not a coding agent. It's a general-purpose autonomous production system. Software engineering was the first vertical. Media production (StoryForge) was the second. The same architecture — daemon + YAML workflows + MCP servers + multi-model routing + persistent state — handles any multi-phase workflow that requires coordination, quality gates, and autonomous execution.

## The Pattern

Every pipeline above follows the same AO architecture:

```
Persistent State (DB/Git)  →  MCP Servers  →  YAML Workflows  →  Quality Gates  →  Output
(truth)                       (enforcement)    (orchestration)     (human/AI review)  (deliverable)
```

- **Daemon** runs it 24/7 without human intervention
- **YAML** defines the phases, agents, and quality gates
- **MCP servers** connect agents to external systems and enforce constraints
- **Multi-model routing** picks the right AI for each phase
- **Quality gates** ensure nothing ships without review
- **Persistent state** maintains continuity across runs

Code was the first vertical. Media was the second. The architecture is vertical-agnostic.

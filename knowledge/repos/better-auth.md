# better-auth

**Repo**: `launchapp-dev/better-auth`
**Visibility**: Public (open-source)
**Language**: TypeScript
**Version**: 0.0.2-beta.8 (root); packages published to npm
**License**: MIT
**Last updated**: 2025-04-18

## Purpose

The most comprehensive authentication framework for TypeScript. Framework-agnostic auth and authorization library providing comprehensive features out of the box: 2FA, multi-tenant support, OAuth, sessions, and a plugin ecosystem.

## Tech Stack

- TypeScript monorepo
- pnpm workspaces + Turborepo
- Biome (linting/formatting)
- bumpp (versioning)

## Key Features

- Framework-agnostic (works with any TypeScript backend)
- Plugin ecosystem for advanced features
- 2FA, multi-tenant, OAuth, sessions
- Focus on "doing better" than existing auth libraries

## Published Packages

Packages published to npm under the `better-auth` namespace (exact package names depend on monorepo structure).

## Dependencies on Org Products

None (standalone, open-source)

## What Depends On This

- `launchpad-baas` (core auth system)
- `launchapp-lite` (auth package)
- `launchpad-saas-template` (via `@launchpad/auth`)
- `launchpad-auth-sdk` (wraps better-auth for Launchpad)

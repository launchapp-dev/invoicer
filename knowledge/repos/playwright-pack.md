# playwright-pack

> Claude Code plugin pack: Playwright

## Purpose

Browser automation, E2E testing, screenshots, accessibility audits

## Visibility: Private
## Maturity: Active Development
## Last Updated: 2026-03-16

## Slash Commands

| Command | Purpose |
|---------|---------|
| `/accessibility-audit` | Run an accessibility audit on a page |
| `/performance-audit` | Audit page performance and load metrics |
| `/record-flow` | Record a user interaction flow for test generation |
| `/scrape-page` | Scrape structured data from a web page |
| `/screenshot` | Take a screenshot of a URL or page element |
| `/test-flow` | Run an E2E test flow against a URL |

## Subagents

| Agent | Purpose |
|-------|---------|
| `form-tester` | Automate form submission testing across multiple inputs |
| `visual-regression` | Detect visual regressions by comparing screenshots |

## Installation

```
claude plugin add launchapp-dev/playwright-pack
```

# launchpad-i18n

> Internationalization utilities covering translation, formatting, and pluralization built on native `Intl` APIs.

## Purpose

Provides translation with interpolation, pluralization, and context support, alongside number formatting, date formatting, relative-time formatting, and list formatting. Relies entirely on the platform's built-in `Intl` APIs. Includes a memory store and static loader for translation resources.

## Visibility: Private
## Maturity: Single commit. Feature-complete for its scope.

## Tech Stack

- TypeScript
- Memory store
- Static loader
- `Intl` APIs (no external runtime dependencies)

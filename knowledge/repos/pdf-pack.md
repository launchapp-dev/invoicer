# pdf-pack

> Claude Code plugin pack: PDF

## Purpose

PDF generation, parsing, form filling, OCR, document processing

## Visibility: Private
## Maturity: Active Development
## Last Updated: 2026-03-17

## Slash Commands

| Command | Purpose |
|---------|---------|
| `/extract-tables` | Extract tables from a PDF into structured data |
| `/extract-text` | Extract raw text content from a PDF |
| `/fill-form` | Fill PDF form fields programmatically |
| `/merge-pdfs` | Merge multiple PDF files into one |
| `/pdf-summary` | Generate a summary of a PDF document |
| `/pdf-to-markdown` | Convert a PDF document to Markdown format |

## Subagents

| Agent | Purpose |
|-------|---------|
| `contract-reviewer` | Review contracts for key clauses and risks |
| `document-analyzer` | Analyze and extract structured data from documents |

## Installation

```
claude plugin add launchapp-dev/pdf-pack
```

# offline-sdk

> Offline-first React capabilities — network/sync state hooks, offline queries/mutations, cache management, and conflict resolution.

## Purpose

Provides React hooks and utilities for building offline-first applications. Covers network and sync state detection, offline queries and mutations, client-side cache management, conflict resolution strategies, and optimistic updates.

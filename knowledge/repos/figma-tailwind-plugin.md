# figma-tailwind-plugin

**Repo**: `launchapp-dev/figma-tailwind-plugin`
**Visibility**: Public
**Language**: TypeScript
**Last updated**: 2025-04-10

## Purpose

Figma plugin that generates Tailwind CSS React components from Figma design variables. Bridges the design-to-code gap for teams using Figma + Tailwind.

## Features

- Reads Figma variables (color tokens, spacing, typography, etc.)
- Generates Tailwind React components based on those variables
- Useful for design system token synchronization

## Tech Stack

- TypeScript
- Figma Plugin API
- Tailwind CSS

## Dependencies on Org Products

Adjacent to `design-system` (`@audiogenius/design-system`) — could be used for generating design token components for the design system.

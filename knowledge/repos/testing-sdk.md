# testing-sdk

> Testing utilities for Launchpad apps — TestProvider, mock server, auth state helpers, assertions, and debug tools.

## Purpose

Provides a suite of testing utilities for applications built on the LaunchPad platform. Includes `TestProvider`/`TestWrapper` components, a mock server with request history, auth state helpers, common error handlers (401/403/404/500), wait-for utilities, mock factories, assertion helpers, cleanup utilities, and debug tools.

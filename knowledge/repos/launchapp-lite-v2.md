# launchapp-lite-v2

**Repo**: `launchapp-dev/launchapp-lite-v2`
**Visibility**: Private
**Language**: TypeScript
**Last updated**: 2026-03-17

## Purpose

Version 2 of the LaunchApp Lite SaaS starter template. Iterates on `launchapp-lite` with Tailwind CSS v4 and refinements. Developed in parallel with v1.

## Tech Stack

Same as `launchapp-lite`:
- React Router v7 (Framework Mode)
- Hono API server
- Better Auth
- Drizzle ORM
- Supabase / PostgreSQL
- Stripe (payments)
- Tailwind CSS v4 (upgrade from v1)
- React Native / Expo (mobile)
- pnpm + Turborepo

## Dependencies on Org Products

- `better-auth` (org's own auth library)
- Drizzle ORM, Stripe, Supabase (external)

## Relationship to v1

Appears to be iterating in parallel with `launchapp-lite`. The v2 designation likely reflects the Tailwind CSS v4 upgrade and associated template refinements rather than a breaking rewrite.
